# HR Analytics Dashboard

## Overview
- Analyzed HR data to understand employee attrition, performance, and workforce trends for better decision making.

## Tools
- Power BI | Excel | SQL

## Dataset
- HR employee dataset containing demographics, job roles, salary, and attrition details.

## Work Done
- Cleaned and prepared HR data  
- Analyzed attrition and workforce metrics  
- Built interactive Power BI dashboard  

## Key Insights
- Identified factors influencing employee attrition  
- Analyzed department-wise and role-wise trends  
- Highlighted key HR performance metrics  

## Output
- Interactive Power BI HR Analytics Dashboard.
